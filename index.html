# RSI سکینر ویب پروجیکٹ یہ ایک سادہ ویب پروجیکٹ ہے جو اردو زبان میں RSI سکینر کے سگنلز دکھاتا ہے۔ ## فائلز کی تفصیل - `index.html` : ویب صفحے کا بنیادی ڈھانچہ اور اردو زبان کی ترتیب۔ - `styles.css` : صفحے کی اسٹائلنگ کے لیے CSS۔ - `script.js` : سادہ جاوا اسکرپٹ جو فرضی RSI سگنلز دکھاتا ہے۔ - `README.md` : پروجیکٹ کی تفصیل اور استعمال کا طریقہ۔ ## استعمال کا طریقہ 1. تمام فائلز کو ایک فولڈر میں رکھیں۔ 2. `index.html` کو کسی بھی جدید براؤزر میں کھولیں۔ 3. آپ جاوا اسکرپٹ فائل `script.js` میں اپنا حقیقی ڈیٹا یا API استعمال کر سکتے ہیں تاکہ سگنلز کو خودکار بنایا جا سکے۔ ## نوٹس - یہ پروجیکٹ صرف تعلیمی اور ڈیمو کے لیے ہے۔ - مزید ترقی کے لیے آپ کو جاوا اسکرپٹ میں ڈیٹا فچنگ اور پراسیسنگ شامل کرنا ہو گی۔ --- اگر آپ کو مزید مدد چاہیے ہو تو بلا جھجھک رابطہ کریں۔
